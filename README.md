# Movie Explorer

Учебное React-приложение для поиска и просмотра информации о фильмах.

Проект сделан как итоговая работа по дисциплине «Веб-разработка и дизайн интерфейсов». В приложении можно смотреть популярные фильмы, искать фильмы по названию, открывать отдельную страницу фильма и добавлять фильмы в избранное.

## Что есть в проекте

- главная страница с описанием приложения;
- страница со списком популярных фильмов;
- страница поиска фильмов;
- отдельная страница фильма с описанием, рейтингом и ссылкой на Кинопоиск;
- страница избранного;
- сохранение избранных фильмов в localStorage;
- Storybook для UI-компонента FilmCard.

## Технологии

- React
- React Router
- Redux Toolkit
- Material UI
- Storybook
- ESLint
- Prettier
- Vite

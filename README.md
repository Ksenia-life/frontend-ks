# Movie Explorer

Учебное React-приложение для поиска и просмотра информации о фильмах.

Проект сделан как итоговая работа по дисциплине «Веб-разработка и дизайн интерфейсов». В приложении можно смотреть популярные фильмы, искать фильмы по названию, открывать отдельную страницу фильма и добавлять фильмы в избранное.

## Ссылки

- Деплой: https://ksenia-life.github.io/frontend-ks/

## Функциональность

В приложении реализованы:

- главная страница с описанием проекта;
- страница со списком популярных фильмов;
- страница поиска фильмов по ключевому слову;
- фильтрация фильмов по жанру;
- отдельная страница фильма с описанием, рейтингом и дополнительной информацией;
- переход на оригинальную страницу фильма на Кинопоиске;
- добавление и удаление фильмов из избранного;
- отображение количества избранных фильмов в меню;
- UI-kit компонентов в Storybook
